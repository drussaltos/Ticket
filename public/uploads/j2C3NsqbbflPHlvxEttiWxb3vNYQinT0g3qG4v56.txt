2. СИСТЕМНЫЕ ТРЕБОВАНИЯ
=======================

Минимальная конфигурация
------------------------

Windows 98/Me/2000/XP; Pentium IV 1,5 ГГц; ОЗУ 256 Мб; Видео: NVidia GeForce3, ATI Radeon 7500; DirectX 8.1; 2,5 Гб на жестком диске; DirectX совместимая звуковая карта.


Рекомендуемая конфигурация
--------------------------

Windows 98/Me/2000/XP; Pentium IV 2,5 ГГц; ОЗУ 768 Мб; Видео: NVidia GeForce FX5900, ATI Radeon 9800; DirectX 8.1; 3,5 Гб на жестком диске; DirectX совместимая звуковая карта.

Для установки игры необходимо иметь не менее 2,5 Гб свободного места на жестком 
диске. Рекомендуется дополнительно иметь не менее 600 Мб свободного места для 
сохранения игр и файла подкачки.


3. УСТАНОВКА ИГРЫ
=================================
Вставьте первый компакт-диск игры в привод CD-ROM. Установка игры начнется 
автоматически. 

З а м е ч а н и е. Если в вашей системе отключена опция автоматического  
распознавания компакт-дисков (Auto insert notification), меню  автозапуска  не  будет автоматически появляться на экране при вставке компакт-диска в привод CD-ROM.  В этом случае вам следует вручную запустить файл Setup.exe из корневой папки первого компакт-диска игры.

По окончании установки, вы можете запустить игру либо из стартового меню 
компьютера, либо из директории установки игры, файл aim2.exe.

Перед запуском игры следует вызвать утилиту настроек из программной группы 
«Механоиды 2» в стартовом меню. 
Настройки игры описаны в пункте 4.

Примечание 1: при первом запуске игры на экране появится диалог, предлагающий 
ввести код защиты от копирования, напечатанный на компакт-диске игры.
При последующих запусках выполнять эту процедуру не потребуется.

Примечание 2: пожалуйста, учитывайте, что системе защиты диска от копирования 
требуется некоторое время (до 20 секунд, в зависимости от типа дисковода) после 
ввода ключа для идентификации диска. 
Не пытайтесь в это время запустить игру вновь - это может привести к ошибке. 
Пожалуйста, дождитесь окончания проверки диска и автоматического запуска игры.

Примечание 3: CD-версия игры для запуска требует наличия CD 3 в дисководе CD-ROM, DVD-версия игры требует наличия диска DVD в дисководе DVD-ROM.


4. СОВЕТЫ ПО НАСТРОЙКЕ ИГРЫ
===========================
Задание конфигурационных параметров игры выполняется в игре через меню «Настройки». 

Настройка параметров графики, звука, управления и сети осуществляется в соответствующих разделах меню.

Кнопка «По умолчанию» устанавливает наиболее общие параметры в активном разделе.
Сохранение измененных параметров всех разделов выполняется по кнопке «Применить».
 
Изменение некоторых параметров возможно только при перезапуске игры.

6. ИЗВЕСТНЫЕ ТЕХНИЧЕСКИЕ ПРОБЛЕМЫ
=================================

Известные проблемы с видеоадаптерами
----------------------------------------

- На видеокартах семейства GeForce в настройках Direct3D должна быть включена «эмуляция таблицы затуманивания» для корректного отображения тумана. В противном случае игра может работать некорректно.

- Игра будет работать медленно, если в системе не установлены драйверы AGP. Если в свойствах видеоадаптера стоит «Тип шины: PCI», значит драйверы AGP не установлены. Эти драйверы должны идти с материнской платой компьютера. Также их можно скачать с сайта-производителя вашей материнской платы.

Если игра работает слишком медленно: 
 - уменьшите качество текстур и дальность видимости;
 - сделайте дефрагментацию диска;
 - попробуйте изменить параметры в настройках видеоадаптера.
- выключите антивирус и остальные программы, которые используют ресурсы компьютера.


Проблемы со звуком
----------------------------------------
При возникновении проблем со звуком рекомендуется установить новые драйверы для звуковой платы или отключить аппаратный ускоритель звука в настройках компьютера.

Перед обращением в службу технической поддержки, пожалуйста, выполните следующие действия:

- В меню «Старт» выберите команду «Пуск». 
- В диалоговом окне наберите «dxdiag» и нажмите «Ввод», чтобы запустить 
Microsoft DirectX Diagnostic Tool. 
- Выполните все тесты. 
- После выполнения всех тестов, нажмите «Сохранить результаты» («Save All 
Information»). 
- Полученный текстовый файл и описание вашей проблемы перешлите в службу 
технической поддержки

- название игры
- версию игры (установленные обновления)
- операционная система
- марка, тип и тактовая частота процессора
- объем оперативной памяти в системе
- тип звуковой карты
- модель и параметры видеоадаптера
- тип привода CD-ROM
- тип мыши и версия драйвера
- версия DirectX
- подробное описание проблемы, с которой вы столкнулись
